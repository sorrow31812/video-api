# 影片討論平台
### 簡介
相關套件：
後端使用nodejs v14 Express + MongoDB + Redis，前端用vue，前後端之間以RESTful api串接
雲端伺服器上以docker、docker-compose運行，並使用bitbucket pipeline做cicd
